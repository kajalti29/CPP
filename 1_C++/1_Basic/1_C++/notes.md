### тЬЕ Example 4: If-Else (Check Even/Odd)

```cpp
#include<iostream>
using namespace std;

int main(){
    int n;
    cout << "Enter a number: ";
    cin >> n;

    if(n % 2 == 0){
        cout << n << " is Even";
    } else {
        cout << n << " is Odd";
    }

    return 0;
}
```

**Output:**

```
Enter a number: 9
9 is Odd
```
